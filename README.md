The **Banking Management System** is a **C-based console application** that simulates basic ATM operations such as cash deposit, withdrawal, balance inquiry, and account records.

This project demonstrates the use of **file handling, control structures, structures, and procedural logic** in C programming to mimic the working of an Automated Teller Machine (ATM).

The system supports multiple users/accounts, each with a unique account number and PIN. It uses text files to persist data such as transaction records, deposits, withdrawals, and balance information.
